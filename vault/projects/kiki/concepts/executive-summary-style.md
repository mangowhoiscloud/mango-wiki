# Executive Summary Style

A communication pattern observed in senior leadership ([[jpark-cfo]]).

## Pattern

Two-layer structure:
1. **Executive Summary** — Top-level findings, 3-5 bullet points max
2. **Backup Data** — Detailed tables, charts, raw numbers underneath

Always includes risk scenarios alongside recommendations. Never presents a single option without alternatives.

## When It Applies

- Board deck preparation
- Cross-functional status updates
- Budget approval requests (always with ROI/Payback Period)
