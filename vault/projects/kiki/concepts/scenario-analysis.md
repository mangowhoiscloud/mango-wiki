# Scenario Analysis

Standard methodology used by the [[finance-team]] for all projections.

## Three-Scenario Model

Every financial projection must include:
1. **Bear** — Pessimistic case
2. **Base** — Most likely case
3. **Bull** — Optimistic case

Each scenario includes sensitivity analysis on key variables (MRR, NRR, Churn Rate, LTV/CAC).

## Usage

- Requested by [[jpark-cfo]] for all board-level decisions
- Produced by [[skim-analyst]] with data source citations
- Required before any investment decision over $10K
