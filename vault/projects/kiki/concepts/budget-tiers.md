# Budget Tiers

Token budget classification system for engineering tasks.

| Tier | Max Tokens | Action |
|------|-----------|--------|
| **S** | 30K | Lead implements directly (config, typos) |
| **M** | 100K | Lead implements directly (single-domain) |
| **L** | 250K | Lead creates sub-issues → App Specialist |
| **XL** | 500K | Must decompose into M-sized pieces first |

XL tasks are never executed as-is — they must be broken down. This prevents runaway token spending.
