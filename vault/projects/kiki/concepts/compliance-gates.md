# Compliance Gates

Mandatory checkpoints in financial workflows where regulatory compliance must be verified before proceeding.

## Key Rules (observed from [[hlee-accountant]])

- 100% checklist completion required before sign-off
- Regulation clause numbers must be cited
- Formal audit trail with approval dates
- No transaction proceeds without documentation verification

## Application

Used in monthly close, AP/AR reconciliation, and any financial transaction over compliance thresholds.
