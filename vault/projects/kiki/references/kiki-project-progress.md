# Kiki Project Progress

Kiki is a Slack observation-based user work-style profiling system that feeds into Paperclip agent optimization.

## Key Milestones

- Initial scaffold with harness + code repo
- Slack MCP server (6 tools)
- Paperclip Plugin (manifest, worker, sync)
- Finance team scenario validated (3 agents, cross-referencing confirmed)
- Engineering team template (hub-spoke, token-optimized)
- LLM Wiki integration (delta tracking, observation log, Obsidian export, health check)

## Skills

| Skill | Purpose |
|-------|---------|
| kiki-setup | Full Paperclip bootstrap + team creation |
| kiki-observe | Slack → behavioral signal extraction |
| kiki-refresh | Profile update + health check |
| kiki-advise | Profile → agent directives |
| kiki-export | Memory → Obsidian wiki |
